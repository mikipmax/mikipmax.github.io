Reseña

Este proyecto se trata en realidad de la migración de un sistema antiguo y rudimentario para realizar estudios actuariales, enfocándose en la optimización de tiempo, dando paso a que abarque más clientes y así se logré expandir el negocio, cuyo fin es crear un producto, en este caso un nuevo sistema que automatice todo el proceso y sea amigable con el usuario final (Actuario). Como detalle técnico se puede decir que, se hace uso de las siguientes tecnologías: Spring boot, lombok, Joinfaces, Primefaces, Postgres, Heroku.
